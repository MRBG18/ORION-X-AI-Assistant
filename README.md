## 🧠 Overview

**ORION X** is a modular AI-powered assistant that understands natural language and executes real-world tasks using:

- Machine Learning  
- Natural Language Processing (NLP)  
- Automation Systems  

It supports multiple interaction modes:

- 🖥️ GUI (PySide6)  
- 💻 CLI (Console-based interaction)  
- 🎤 Voice-based commands  

---

## ⚡ Key Features

### 🗣️ Natural Language Interaction
- Understands commands like:
  - “open chrome”
  - “search python tutorials”
  - “remind me in 5 minutes”
